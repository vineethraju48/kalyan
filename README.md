# Kubernetes Deployment using Terraform for Axivion

**Refer below terraform and kubernetes commands**
  
**To install plugins :** 
```terraform plan```

**To validate the code :** 
```terraform validate```

**Dry run :** 
```terraform plan```

**To deploy :**
```terraform apply```

**To update state file :** 
```terraform refresh```

**To terminate resources :** 
```terraform destroy```

**To get Cluster IP :** 
```kubectl config view```

**To get deployments :** 
```kubectl get deployments```

**To get pods :** 
```kubectl get pods```
